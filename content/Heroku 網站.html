Heroku 網站

先建立自己 Heroku 的帳號，登入後建立第一個 app

下載 Heroku CLI 程式檔案，解壓縮至隨身碟裡面

修改命令搜尋路徑，將 heroku.exe 設置在 date

把 stare_ivp6 拉進小白窗，複製進去

REM for heroku login
set HTTP_PROXY=http://[2001:288:6004:17::53]:3128
set HTTPS_PROXY=http://[2001:288:6004:17::53]:3128

然後重啟小黑窗

先打 cd tmp

再來 cd wcm2021

接著 heroku login -i

之後打上你的 heroku 帳後及密碼

然後 heroku login -a a40923101 (a40923101建立app的名稱)

最後 git push heroku

Heroku倉儲上傳因為不是連接學校網路，所以在heroku那兩行前面加上REM

按下 Save

到小黑窗打

cd tmp

cd wcm2021

git branch

git push heroku

就可以看到倉儲同步
